# 제조 AI-Agent 자율공장 플랫폼

제조 AI-Agent 기반 자율공장 플랫폼을 위한 프론트엔드 애플리케이션입니다.

## 주요 기능

### 1. ChatPanel (LLM 명령 인터페이스)
- GPT 스타일 자연어 명령 입력창
- 명령 입력 시 타이핑 애니메이션과 함께 응답 출력
- 자동 스크롤, 반응형 UI, 모바일 대응
- 명령 기록 유지, 최근 명령 히스토리 표시

### 2. AgentGraph (AI-Agent 협업 시각화)
- LLM Agent 중앙, 도메인 에이전트 원형 배치
- 판단 중/협업 중/완료 상태마다 선 연결 및 색상 변화
- `framer-motion` 기반 상태 변화 애니메이션
- Agent 클릭 시 상세정보, 설명 팝오버 표시

### 3. KPICharts
- 생산량, 불량률, 가동률 등 주요 KPI 차트
- `Recharts` 기반, 깔끔한 툴팁, 애니메이션 효과
- 일/주/월 단위 필터 기능 포함

### 4. FactoryMap (설비 상태 맵)
- 설비별 가동 상태 (정상/경고/이상) 색상 표시
- 클릭 시 상세 센서 데이터 모달 출력
- 확대/축소, 마우스 호버 인터랙션 포함
- @react-three/fiber 기반 3D 공정 맵
- 주요 구성 요소:
  - 설비 3D 모델 (box, cylinder 등 추상화)
  - 생산 라인 흐름 (line, arrow, movement)
  - 클릭 가능 영역: 설비 클릭 시 모달 열림
  - 각 설비 위에 상태 표시 뱃지: "정상", "경고", "이상"
  - `OrbitControls`로 회전/확대 지원

### 5. ReportPreview (PDF 스타일 요약)
- 요약 보고서 모달 (PDF 느낌의 HTML)
- ChatPanel → "보고서 만들어줘" 명령 시 자동 생성
- 저장 또는 출력 버튼 포함

### 6. EventTimeline
- 에이전트 판단 이벤트 로그 실시간 출력
- 가장 최근 이벤트가 상단에 오도록 정렬
- 로그 수준(Info/Warning/Error) 별 색상


## 시작하기

개발 서버 실행:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인합니다.

## 디자인 가이드

- 다크 모드 기본 (배경: `#1E1E1E`, 텍스트: `#F9F9F9`)
- 포인트 컬러: `#EB6100` (브랜드 색상)
- 프리텐다드 웹폰트 기본 적용
- 반응형 레이아웃 (모바일, 태블릿, 데스크톱)
- 상태 변경 및 데이터 업데이트 시 부드러운 애니메이션 효과
